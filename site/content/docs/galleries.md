---
title: "Galleries"
galleries:
    - name: gallery-one
      images:
        - images/*.jpg
---

Pages can contain several galleries. You have to define them on the page metadata:

```markdown
galleries:
    - name: gallery-one
      images:
        - docs/images/*.jpg
```

Then add to the page like this:

<!-- ```markdown -->
<!-- !![gallery-one] -->
<!-- ``` -->
And that's how it will look like:

!![gallery-one]

Great, right?
